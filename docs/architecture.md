# Architecture

The system is personal-first: the Protomentat and Erasmus are the initial dyad. Contracts avoid hard-coding this dyad into the machinery so future isolated dyads remain possible.

## Planes

1. Cognitive gateway.
2. Mission engine.
3. Deterministic-first capability manifold.
4. Landsraad governance.
5. 10th-Man cognitive immune system.
6. Explicit persistence and sleep integration.
7. Validated skills and optional Experience LoRAs.

## Deterministic-first cascade

Rule → deterministic tool → statistics/classical ML → specialist agent → specialist adapter → Protomentat.

Most events should terminate at the cheapest competent layer.

## Persistence

- RAG: explicit content and provenance.
- Ledger: beliefs and contradictions.
- Experience: candidate behavioral lessons.
- Immune memory: anomaly signatures, mitigations, false positives, and wake-up state.
